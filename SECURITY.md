# Política de Segurança

## Versões Suportadas

Apenas a versão estável mais recente (latest release) de nossos projetos recebe atualizações de segurança.

## Reportando uma Vulnerabilidade

Não reporte problemas de segurança através de Issues públicas.

Se você descobriu uma vulnerabilidade de segurança, por favor envie um e-mail para **<renan.fernandes@outlook.com>**.

Você receberá uma resposta em até 48 horas.
